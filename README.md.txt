# NotOrtalaması

Basit not ortalaması hesaplama Java Swing uygulaması.

---

## Kullanılan Teknolojiler

- Java Swing
- NetBeans IDE

---

## Proje Açıklaması

Bu proje, kullanıcıdan isim, öğrenci numarası, vize, final ve proje notlarını alarak ortalamayı hesaplar ve sonucu gösterir.

---

## Nasıl Çalıştırılır?

1. NetBeans'te projeyi açın.
2. NotOrtalaması.java dosyasını çalıştırın.
3. GUI üzerinden notları girip sonucu görün.
